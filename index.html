Bluetooth

Variable Bit Rate para codec AAC. Mejor calidad en auriculares AAC sin el overhead de LDAC

Control de frames para AAC. Mejora sincronización y reduce artefactos en la transmisión
LDAC ABR
Adaptive Bit Rate para LDAC. Reduce cortes ajustando la calidad según la estabilidad de la señal BT

Red y TCP
Congestión TCP
BBR mejora latencia y throughput en LTE/5G. BBR+ más agresivo. Cubic es el default del kernel
TCP Fast Open
Reduce la latencia del handshake TCP enviando datos en el primer SYN. Necesario para BBR
ECN
Explicit Congestion Notification. Notifica congestión sin perder paquetes. Bueno con BBR en 5G
TCP Window Scaling
Permite ventanas de recepción superiores a 64 KB. Necesario para aprovechar rmem_max alto
MTU Probing
Detecta automáticamente el MTU óptimo. Evita fragmentación en 5G SA donde el MTU puede variar
TCP SACK
Selective ACK: retransmite solo los paquetes perdidos en lugar de toda la ventana. Básico para LTE/5G
TCP TW Reuse
Reutiliza sockets en estado TIME_WAIT para nuevas conexiones. Reduce el agotamiento de puertos
Recuperación datos
Reconexión automática de datos móviles al perder y recuperar señal
Google Checkin OFF
Desactiva el checkin periódico de Google. Reduce tráfico de fondo y despertares de red innecesarios
Data No Toggle
Evita el toggle de datos durante reconexiones del modem. Reduce cortes en cambios de celda
Radio Power Save OFF
Desactiva el ahorro de energía adicional del modem. Señal más estable a costa de algo de batería
Kernel VM
VFS Cache Pressure
Agresividad al liberar caché de filesystem. 50 = retiene más caché (apps abiertas más rápido). 100 = default kernel. 200 = descarga caché agresivamente (más RAM libre)
100
Dirty Ratio
% de RAM con datos sin escribir al UFS antes de forzar flush. 10 = flush frecuente (más escrituras, menor riesgo de pérdida). 20 = recomendado (balance). 40+ = agresivo (mejor throughput, más riesgo ante corte de luz)
20

Prefetch predictivo de I/O. El sistema aprende los patrones de carga de cada app y pre-lee datos antes de que los necesites
